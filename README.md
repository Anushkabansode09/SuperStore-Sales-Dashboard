# SuperStore Sales Dashboard (Power BI)

## Overview
This project is an interactive Power BI dashboard built using SuperStore sales data. It provides insights into sales performance, profit trends, customer segments, and regional distribution.

## Key Features
- Sales, Profit, and Order analysis
- Monthly trends visualization
- Region-wise performance (Map)
- Segment and Category breakdown
- Payment mode analysis
- Interactive filters for region selection

## Tools Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Visualization Techniques

## Insights
- Consumer segment contributes highest sales
- COD is the most used payment method
- Sales show growth trend towards year-end
- Certain states generate higher profit
